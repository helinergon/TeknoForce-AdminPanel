# TeknoForce Admin Panel

TeknoForce platformu için geliştirilen yönetim paneli uygulamasıdır.

## Kullanılan Teknolojiler
- ASP.NET Core
- C#
- HTML / CSS / JavaScript

## Amaç
Site yönetimi, içerik kontrolü ve yönetici işlemlerinin tek panelden yapılması.

## Durum
Aktif olarak geliştirilmektedir.
